Adding it to WhatsApp

  1. Download the sticker Immey makes for you — it's already sized to WhatsApp's 512×512 spec.
  2. Open a chat in WhatsApp → tap the sticker icon → the + tray → Create, and pick the file from your gallery.
  3. Prefer a dedicated sticker pack? Import the downloaded file into an app like Sticker Maker or Sticker.ly, which adds an "Add to WhatsApp" button for you.
